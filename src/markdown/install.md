# 安装

打开终端运行下列命令：

```
npm install spring-ui
```

或

```
yarn add spring-ui
```
# 使用

代码示例：

```
<template>
  <div>
    <Button>按钮</Button>
  </div>
</template>
<script>
import {Button, Tabs, Tab, Switch, Dialog, openDialog} from "spring-ui"
export default {
  components: {Button}
}
```
</script>
